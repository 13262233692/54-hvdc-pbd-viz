## 1. 产品概述

特高压直流输电换流站控制中心实时超高频三维局放定位可视化监控大屏——基于 3D MUSIC 空间谱估计算法与 WebGL 体积渲染，在三维空间中实时重现变压器内部绝缘击穿处的局放电弧电场云图，为运维人员提供秒级局放定位与故障预警能力。

- 目标用户：换流站运维工程师、电力调度监控人员、设备状态诊断专家
- 核心价值：将传统 2D 散点图升级为三维体积渲染电场云图，实现皮秒级时差定位精度与亚秒级刷新率，显著提升局放故障识别效率

## 3. 核心流程

**实时局放定位主流程**：多路 UHF 探针持续采集超高频电磁信号 → Node.js 后端通过高性能网络套接字并发接收原始二进制数据流 → 按字节协议拆解出脉冲时钟戳、通道间纳秒时差（TDOA）、能量幅值 → 通过 WebSocket 实时推送至前端 → 前端构建多通道时差矩阵 → 动态求解传感器阵列协方差矩阵并特征值分解 → 运行 3D MUSIC 空间谱估计算法生成三维能量概率矩阵 → 作为 DataTexture3D 上传 GPU → 片元着色器 Raymarching 体积渲染 → 屏幕呈现变压器内部局放电弧 3D 电场云图。

### 4.3 响应式设计

- 桌面优先设计，目标分辨率 3840×2160（4K 大屏）及 1920×1080
- 3D 渲染区自适应占据可用空间
- 数据面板支持折叠/展开，小屏时自动收起侧边栏
- 示波器条带支持垂直/水平布局切换

### 4.4 3D 场景设计指引

- **环境/氛围**：深空黑底、微弱环境光散射、变压器外壳半透明线框
- **光照**：单一冷色方向光 + 电弧云团自发光（点光源动态绑定至局放定位点）
- **相机**：透视相机，45° 俯角初始位置，OrbitControls 支持旋转/缩放/平移
- **构图与焦点**：变压器线框居中、局放电场云为视觉焦点、云团亮度和大小随放电强度动态变化
- **交互与动画**：鼠标悬停高亮云团区域、剖面切割交互、云团脉动呼吸动画
- **后处理**：Bloom 辉光效果（电弧云团发光）、Vignette 暗角、色彩校正
- **性能预算**：目标 30fps @4K、空间谱网格 64³ 分辨率、GPU 显存占用 <512MB
